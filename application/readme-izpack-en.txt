
The MyCoRe DocPortal Project
============================

Welcome!

Thank you for your interesting of the MyCoRe DocPortal Project.
With this installation you get a complete running system of a 
DocPortal database. This application is a demonstration of the 
program. If you want to install a server system, please read the 
documentation of the project.

After this installation do the following:

1. Start the script papyri/bin/hsqldbstart.xxx (xxx is depended on your OS)
2. Start the script papyri/bin/jettystart.xxx (xxx is depended on your OS)
3. Run a web browser with the URL http://localhost:9191/

Alternate you can use the ShortCut entries under Start-->Program.

Have a lot of fun!
