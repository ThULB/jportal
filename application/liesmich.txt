README
======

MyCoRe ist ein Open Source Projekt zur Entwicklung eines Systems für Digitale
Bibliotheken und Archivlösungen. Die technische Basis des Systems bilden
Java-Klassenbibliotheken, XML-Techniken und unterschiedliche Datenbank-
Backends wie z.B. HSQLDB, IBM DB2 und MySQL.
Die Distribution besteht aus zwei Teilen:

MyCoRe
======

Der MyCoRe "Kern" beinhaltet eine Reihe Module, die grundlegende
Funktionen zur Bildung von digitalen Archiven bereitstellen. Er ist gleich für
alle lokalen Anwendungen und unterstützt verschiedene Datenbank Backends,
sowohl kommerzielle wie den IBM DB2 als auch freie Open Source Datenbank MySQL.

DocPortal
=========

DocPortal ist die mitgelieferte Beispielanwendung eines Dublin Core
Dokumentservers, der auf dem MyCoRe-Kern basiert.
Die Anwendung implementiert ein Archiv für digitale, multimediale
Dokumente und kann als Publikationsserver verwendet werden. Es ist kein
komplettes Produktionssystem, beinhaltet aber alle grundlegenden Funktionen
für einen Publikationsserver, wie webbasierte Recherchefunktionen, Editoren
und einen einfachen statischen Workflow, sowie eine Nutzerverwaltung und
Rechteverwaltung.
Das Verzeichnis 'documentation' beinhaltet eine Beschreibung der DocPortal-
Anwendung. 

Einfache Installation
=====================

Sind Java 1.5, Ant 1.7 und ein svn-Client auf Ihrem Rechner installiert, können
Sie mit wenigen Schritten ein MyCoRe/DocPortal installieren. Die Installation holt 
alle erforderlichen Daten und Programme für eine Beispielanwendung von www.mycore.de. 
Zur Speicherung der Metadaten wird HSQLDB, für die Inhalte (Bilder, Texte, ) das lokale 
Dateisystem verwendet. Und hier sind die durchzuführenden Schritte:

0. Legen Sie ein Verzeichnis mit dem absoluten Pfad <pfad> an und führen Sie in dem Verzeichnis die folgenden
   Kommandos aus:

1. svn export http://server.mycore.de/svn/mycore/trunk/install.xml

2. ant -Ddestdir=<pfad> -Drevision=HEAD -f install.xml

Das zweite Kommando dauert etwa 10 Minuten. Die Installation ist erfolgreich, wenn sich im Verzeichnis
<pfad> die Datei homepage.html befindet.


Start von DocPortal
===================

 - wechseln Sie ins Verzeichnis <pfad>/docportal/build/bin und rufen Sie unter Windows die Kommandos

   hsqldbstart.cmd und jettystart.cmd auf

   und unter Linux
   hsqldbstart.sh und jettystart.sh

Geben Sie als Adresse in einem Browser http://localhost:8291 ein

Stop von DocPortal
==================

 - wechseln Sie ins Verzeichnis <pfad>/docportal/build/bin und rufen Sie unter Windows die Kommandos

   jettystop.cmd und hsqldbstop.cmd aus

   und unter Linux
   jettystop.sh und hsqldbstop.sh



MyCoRe wurde unter verschiedenen Plattformen mit unterschiedlichen Datenbankbackends getestet. 

+ AIX5.2 mit DB2 und Lucene
+ Linux mit HSQLDB und Lucene
+ Windows XP mit HSQLDB und Lucene
+ Linux mit MySQL und Lucene
+ Windows 2000 mit MySQL und Lucene

Weitere Informationen zur Installation finden Si unter documentation/INSTALL.DE.*.
