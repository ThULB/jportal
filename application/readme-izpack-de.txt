
Das MyCoRe DocPortal Projekt
============================

Willkommen!

Danke, dass Sie sich fuer das MyCoRe DocPortal Projekt interessieren. 
Mit der folgenden Installation erhalten Sie eine vollstaendige Anwen-
dung fuer Ihr Zielsystem. Die Applikation ist fuer Demonstrationszwecke 
gedacht. Sollten Sie eine eigene Instanz dieses Projektes installieren 
wollen, so lesen Sie bitte die entsprechende Dokumentation unseres 
Projektes.

Nach dieser Installation muessen Sie folgendes tun:

1. Das Script docportal/bin/hsqldbstart.xxx (je nach OS) starten.
2. Das Script docportal/bin/jettystart.xxx (je nach OS) starten.
3. Im Web-Browser die Adresse http://localhost:8291/ angeben.

Alternativ finden Sie fuer MS Windows auch entsprechende ShortCut-
Eintraege unter Start-->Programme.

Viel Erfolg!
